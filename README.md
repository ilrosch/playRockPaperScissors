# Консольная игра "Камень, Ножницы, Бумага"

## Дайате начнем

- Склонируйте репозиторий

```console
    git clone git@github.com:ilrosch/playRockPaperScissors.git
```

- Установите нужные пакеты и зависимости

```console
    make install
```

- Запустите игру

```console
    make start
```
